System Description

This project focuses on achieving precise indoor positioning by tracking signal variations and heading data. The system captures raw RSSI samples from wireless modules while integrating directional orientation to map movement dynamically within indoor environments. The collected data streams are processed alongside predefined path constraints and optimized through a path-aware fusion model to eliminate signal noise and deliver real-time location tracking.

This project focuses on developing an intelligent indoor navigation system utilizing long-range wireless infrastructure. A mobile node continuously gathers unprocessed RSSI samples and real-time heading data as users navigate through an indoor space. These telemetry streams are filtered through predefined structural path constraints and calculated via a path-aware fusion model to calculate accurate coordinate positions.
